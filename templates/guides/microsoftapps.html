Update Microsoft Applications


Host-by-host basis (Manually)

  1. Go to Microsoft Store on the machine.
  2. Click the Library button at the bottom left corner of the screen, then Get updates. If there are updates, it will start installing automatically.

Bulk updates (Automatic)

  1. Download either one of the microsoftapps scripts here and execute it in the machine's Windows Command Prompt or PowerShell.
Note: Some applications manage their own updates.
If there still are issues with updating Microsoft Applications
Uninstall and reinstall the applications.
