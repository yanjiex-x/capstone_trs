Update Firefox


Host-by-host basis (Manually)

  1. Click the button at the top-right corner of the browser.
  2. In the drop-down menu, choose Help, then About Firefox.
  3. A pop-up window should appear along with an update status message. If the browser is running the latest version, it will say Firefox is up to date. Else, it will say Restart to Update Firefox. Click on that to install the updates and restart the browser.

Bulk updates (Automatic)

  1. Download the firefox script here and execute it in the machine's Windows Command Prompt.
If there still are issues with updating Firefox
Download the installer, and make sure to close Firefox before running the installer.
