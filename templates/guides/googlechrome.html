Update Google Chrome


Host-by-host basis (Manually)

  1. In the address bar, type chrome://settings/help. Or
  2. Click on the icon at the top-right corner of the browser.
  3. Hover over Help and select About Google Chrome.
  4. Google Chrome should automatically start to check for updates. If there are, click Relaunch or simply close and reopen the browser to apply the updates.

Bulk updates (Automatic)

  1. Download either of the googlechrome scripts here and execute it in the machine's Windows Command Prompt or PowerShell.

Configure auto-updates

Using Group Policy

In Group Policy (Computer Configuration folder):

  1. Go to Google > Google Update > Applications
  2. Enable the Update policy override default policy.
  3. Under Options, choose Allow updates (recommended).
  4. Go to Google > Google Update > Applications > Google Chrome and repeat steps 2 and 3 to make sure auto-updates are also always allowed for Chrome browser.
You can optionally override this setting for an individual app by using the Update policy override policy in the specific app folder.
